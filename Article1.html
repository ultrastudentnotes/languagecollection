What is HTML?

HTML stands for HyperText Markup Language. It is the basic building block of a website. Every website you see on the internet is created using HTML.

Why HTML is Important?

HTML is used to structure the content of a webpage. Without HTML, a website cannot exist. It tells the browser what to display.

Basic HTML Structure

<!DOCTYPE html> <html> <head> <title>My Page</title> </head> <body> <h1>Hello World</h1> <p>This is my first webpage</p> </body> </html>

Common HTML Tags

  • <h1> - Heading
  • <p> - Paragraph
  • <a> - Link
  • <img> - Image

Example

This is a simple example:

<h1>Welcome</h1> <p>Learning HTML is easy!</p>

Conclusion

HTML is the foundation of web development. If you learn HTML well, you can build websites easily.
